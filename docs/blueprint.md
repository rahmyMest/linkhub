# **App Name**: LinkHub

## Core Features:

- Link Saving: Save links with associated titles and descriptions either by pasting or direct import via browser bookmark.
- Link Categorization: Categorize saved links using tags or categories for better organization. App should provide a series of default categories that the AI tool will assess when new URLs are added.
- Tag suggestions: Suggest tags via a LLM tool to quickly categorize and organize the links
- Link Browsing: Browse saved links by category, tags, or search keywords.
- Link Editing: Edit the title, description, and category of saved links.
- Automatic Link Preview Generation: Generate previews of saved links for visual identification.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust and organization.
- Background color: Very light blue (#E8EAF6), almost white.
- Accent color: Purple (#9575CD) to provide highlights and interactive cues, ensure accessibility by meeting contrast standards on primary background
- Headline font: 'Space Grotesk' sans-serif for a modern and techy look (headlines).
- Body font: 'Inter' sans-serif (body).
- Simple and clear icons for categories and actions.
- Clean and intuitive layout for easy navigation.
- Subtle animations for a smooth user experience.
